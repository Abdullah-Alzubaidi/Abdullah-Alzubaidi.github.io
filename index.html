Projects

Electronics Repair Services System

The Electronic Repair Services System is an online platform developed using Dart and Flutter. It allows customers to easily request technicians for device repairs and enables shops to manage service requests. The system aims to be efficient, interactive, and successful, offering benefits such as streamlined repair processes, real-time communication, and data analytics for improved services. With Firebase as the database technology, it ensures scalability and synchronization of data. Overall, the system aims to address existing delivery process issues and enhance the repair service experience for both customers and shops.

Hangman

The Hangman game uses Java Swing framework for GUI components like frames, labels, text fields, radio buttons, and buttons. It also leverages Java's built-in graphics by overriding the paint(Graphics g) method for custom drawing, displaying visual elements such as the spaceship and alien for incorrect guesses. This combination creates an interactive and engaging experience for players.

GLIE Internship Website

The GLIE Internship Website, created by our interns in September/October 2023, is a user-friendly platform for students seeking internships. The site features an interactive map showcasing company locations in Malaysia and informative graphs on district and state distributions. Upcoming enhancements include a search function for location and faculty-based company listings and a range-based locator for personalized exploration. Explore the evolving functionalities by selecting pages from the sidebar, designed to streamline the internship exploration process for students.
